RandomEngine v0

As the name suggests: an engine that assigns random scores to objects and random scores to the words that make up an object's meta text.
This engine is part of SABIO because it highlights the idea that tools for the navigation of bias also need to be disruptive (in contrast to being focused on a task). Further, any engine also brings its own biases that might shadow or distract from bias in the collection. The random engine is arguably the least biased and the most disruptive engine possible. We therefore encourage you to play with the random engine as well and to compare what you find to the other engines.

Functionality

Very simple: for each object produces a random score (between 0 and 1). Then, as evidence, it picks random words from the object's texts and assigns random scores to these words.

Interpretation of Outputs

Because all the scores are completely random, the results from this engine have no real interpretation. There is one interesting phenomenon, however, that emerges from randomness: You might notice, that certain objects and words come up more prominently (often) than others; these are the most frequent objects/words in the collection and the random engine "floats them to the top". This is because if everything is chosen at random, very frequent elements naturally have a higher chance of being selected.

Parameters

This engine only has a nonce parameter because the engine actually needs no parameters at all; the nonce parameter therefore has no function.
