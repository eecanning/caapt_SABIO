Typicality Engine v0

The typicality engine uses the probability of the language attached to an object. This probability is in a linguistic sense: If the title or description of an object contains phrases or words that are common in the collection as a whole, this object will have a high linguistic probability.
Probability can be used to capture biases The concept of typicality is derived from probability, namely the most probable phrases of words are not the most typical.
